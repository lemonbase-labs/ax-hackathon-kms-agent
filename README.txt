KMS Agent — 실행 방법 (mac)

[처음 한 번만 — 1단계] 터미널을 열고 아래를 그대로 복사해서 실행:
    curl -LsSf https://astral.sh/uv/install.sh | sh
실행이 끝나면 터미널을 한 번 닫았다 다시 열어 주세요.

[처음 한 번만 — 2단계] 별도로 전달받은 `.env` 파일을 이 폴더(README.txt가 있는 폴더)에 그대로 넣어 주세요.
파일 이름은 반드시 `.env` (앞에 점이 붙은) 그대로여야 합니다.

[매번 실행할 때]
    1. 이 폴더를 Finder에서 엽니다.
    2. 터미널에서 이 폴더로 이동한 뒤 다음을 실행:
           ./start.sh
       (또는 start.sh를 start.command로 이름 변경하면 Finder에서 더블클릭으로 실행됩니다.)
    3. 잠시 후 브라우저에 http://127.0.0.1:8000 화면이 자동으로 열립니다.
    4. 종료하려면 터미널 창에서 Ctrl+C.

문제가 생기면 터미널에 출력된 메시지를 그대로 공유해 주세요.
